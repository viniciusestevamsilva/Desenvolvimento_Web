0 HTML (Hypertext Markup Language) é a base para a criação de páginas web. Utilizando tags como <h1> a <h6> para cabeçalhos e <p> para parágrafos, ele organiza o conteúdo de forma <estruturada>. Além disso, o CSS é responsável pela parte visual das páginas, como cores, fontes e disposição dos elementos. Com o uso de A design responsivo, as páginas se adaptam automaticamente ao tamanho da tela de dispositivos móveis. Por exemplo, a fórmula da água H20 e a equação de Einstein, E = mc2, utilizam subíndices e superíndices para representações científicas. A evolução da web trouxe a necessidade de criar páginas dinâmicas e interativas, com JavaScript. Contudo, algumas práticas antigas, como a criação de páginas estáticas, estão sendo descontinuadas em favor de abordagens mais modernas.
