Estudo sobre Google Fonts

Este é um parágrafo com a fonte Roboto, uma fonte sans-serif moderna e muito usada para textos legíveis em web design.

Este parágrafo usa a fonte Open Sans, que é muito popular por sua legibilidade e estilo clean, ótima para sites corporativos.

Aqui temos a fonte Lora, uma fonte serifada que transmite elegância e é ótima para textos mais longos, como blogs e artigos.

E este parágrafo está usando a Playfair Display, uma fonte serifada clássica que combina bem com títulos e textos estilizados.
